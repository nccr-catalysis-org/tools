List of digital tools built by NCCR Catalysis (under construction)

  • Atom detection : a tool to detect atoms in .tiff microscopy images
  • Spock : generate volcano plots ???
  • Saturn : Sample-efficient Generative Molecular Design using Memory Manipulation
  • Morfeus : calculates molecular features from 3D structures with a focus on steric descriptors. It can be used as a Python library or through command line scripts.
  • CurryBO : Bayesian optimization over curried function spaces
  • Scine : Software for Chemical Interaction Networks
  • Navicat : A platform for catalyst discovery
  • Equireact : Geometric Deep Learning for Chemical Reactions
  • B2R2 : Physics-based representations for machine learning properties of chemical reactions
